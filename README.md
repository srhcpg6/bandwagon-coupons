## 搬瓦工优惠码是什么？先说清楚这个概念

搬瓦工优惠码（BandwagonHost Promo Code）是一种循环折扣码。

所谓"循环"，意思是你首次购买用了某个优惠码，后续每次续费都会自动按相同折扣率计算，不用重新输入。这一点和很多其他平台"只有首购折扣"的玩法不同——对于打算长期用的用户来说，哪怕只有6-7%的折扣，三五年下来也会省出一笔钱。

常规优惠码折扣力度一般在5%到7%之间。大促专属码（如双十一、黑五）力度会更大，去年双十一的 `ILOVEBANDWAGON` 达到了11%。但大促码用完后通常很快失效，不是长期可用的选项。
